# octra-signer-demo

a reference dapp for the popup signer being proposed for [octra-labs/webcli](https://github.com/octra-labs/webcli). live at https://octra-signer-demo.152716.xyz/.

## try it

the popup signer isn't merged upstream yet. until it is, you need to run webcli from the PR branch: https://github.com/defi-failure/webcli

then load the demo and click *connect wallet*.
